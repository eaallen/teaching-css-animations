Simple Animations

You can use the transition property to specify simple animations between actions in CSS.

For example, here I have used transition to make this card get bigger when you hoover the cursor of it. The follow is the example in code

            .change-color {
                transition: 500ms;
            }

            .change-color:hover{
                background-color: #444;
                color: #FDFDFD;
            }
